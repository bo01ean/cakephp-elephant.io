# Elephant.IO Plugin for CakePHP #

for CakePHP 2.x

## Requirement ##

* PHP version: PHP 5.3+
* CakePHP version: 2.x Stable
* Composer

## Installation ##


## Enable plugin ##

In 2.0 you need to enable the plugin in your `app/Config/bootstrap.php` file:

```php
    CakePlugin::load('ElephantIO', array('bootstrap' => true));
```
